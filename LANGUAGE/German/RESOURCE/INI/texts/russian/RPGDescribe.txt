Strength {
 (Energie) Stärke. Wird benötigt, um hart zu treffen, weit zu werfen und viel zu tragen. Beeinflusst Gewichtspunkte (Tragefähigkeit) und Lebenspunkte. Die definierende Eigenschaft für Fähigkeiten, die starken körperlichen Einsatz erfordern, insbesondere schwere und mittlere Waffen sowie allgemeine Waffenfähigkeiten. 
}
Perception {
 (Einsicht) Wahrnehmung. Das Talent für klare Sicht und gutes Gehör. Ohne ein gutes Auge fliegen Kanonenkugeln nur in die ungefähre Richtung des Gegners. Beeinflusst insbesondere Genauigkeit und Heimlichkeit. 
}
Agility {
 (Reaktion) Gewandtheit - Hand-Auge-Koordinierung. Ein unbeholfener Mann verkantet sein Schwert in dessen Scheide und verhakt den Enterhaken in der eigenen Hose. Erforderlich für Fähigkeiten wie leichte Waffen, mittlere Waffen, Pistolen und Entern. Beeinflusst die maximale Energie. 
}
Charisma {
 (Autorität) Charisma. Die Summe aus einem männlich geschnittenen Gesichtsausdruck und einer wohlbalancierten Zunge. Ein Anführer muss keine Gewalt anwenden, um die Waren eines erfolgreichen Händlers in die Hände zu bekommen. Sie werden ihm überreicht! Die besten Leute folgen ihm, und die schönsten Frauen warten in jedem Hafen auf ihn. Führung ist ein notwendiges Attribut für Fähigkeiten wie Autorität und Handel. Sie beeinflusst somit die Zahl der Offiziere, die folgen. 
}
Intellect {
 (Talent) Intellekt. Wissen, Weisheit und die Fähigkeit, sich schnell entscheiden zu können. Ein kluger Charakter hat keine Probleme mit dem schnellen Nachladen einer Waffe oder dem Herunterhandeln im Laden. Die Verwendung von Navigationsinstrumenten erfordert ebenfalls kluge Köpfe. Beeinflusst die Fähigkeiten Handel, Kanonen, Navigation und insbesondere die Schnelligkeit, mit der ein Mann im Rang aufsteigt und neue persönliche Fertigkeiten und Schiffsfertigkeiten lernt. 
}
Endurance {
 (Ausdauer) Konstitution. Dringend erforderlich für einen Helden, um den schwierigen Beruf des Piraten zu überleben. Diese Begabung wird von potenziellen Kunden auch nicht weniger benötigt. Reparatur und Verteidigung werden von dieser Begabung am meisten beeinflusst, ebenso wie die Steigerung der Lebenspunkte und das Gewichts. 
}
Luck {
 (Erfolg) Glück - Was soll man noch über Glück sagen? Wenn es Glück gibt, dann sind wir ehrenwerte Männer. Wenn nicht, dann kommen wir zu nichts. Hilft, wenn sonst nichts mehr hilft. Eine Kugel fliegt an Ihnen vorbei und ein Gegner geht an Ihnen vorbei, ohne Sie zu bemerken. Beeinflusst Fähigkeiten mit Pistolen, Glück und Heimlichkeit. 
}
FencingLight {
 Fertigkeit für leichte Waffen - vor allem Rapiers und kleine Mordwaffen. Hängt von Geschicklichkeit und Einsicht ab. (G*0,9+E*0,1) 
}
Fencing {
 Fertigkeit für Säbel - Waffen, die nicht nur Stärke, sondern auch eine bestimmte Fähigkeit erfordern. Hängt von Kraft und Reaktion ab. (K*0,6+R*0,4) 
}
FencingHeavy {
 Verwendung von schweren Breitschwertern, Äxten und Schwertern erfordert viel physische Stärke. Hängt von Kraft und Ausdauer ab. (K*0,9+A*0,1) 
}
Pistol {
 Die Fertigkeit, technisch fortschrittliche Geräte zu verwenden, mit denen kleine Bleikugeln über große Entfernungen geschossen werden. Erfordert Geschick und ein bisschen Glück. Wer weiß schon, wo die Kugel hinfliegt ... Hängt von Reaktion und Erfolg ab. (R*0.5+E*0.5) 
}
Fortune {
 Geschick - Eigentlich ein gutes Los. Wird immer gebraucht. Hängt vom Erfolg ab. (S) 
}
Leadership {
 Führung. Wenn alle erkennen, dass sie mit Ihnen nicht in jedem Konflikt untergehen, wächst ihr Glaube in Sie und die Schenken füllen sich mit jenen, die sich Ihnen anschließen möchten. Feindliche Schiffe ergeben sich beim bloßen Anblick Ihrer Flagge. Und wenn Ihre Führung hoch genug ist, würden Ihre Männer sogar den Seeteufel selbst entern. Hängt von Autorität und Talent ab. (A*0,9+T*0,1) 
}
Commerce {
 Handel. Die Fähigkeit, ein Höllengeld mit etwas zu machen, für das Sie noch tags zuvor einen Apfel und ein Ei gezahlt haben. Sie müssen die fetten Handelsrouten kennen, um genaue Informationen übe den Markt zu erhalten, sowie gute Beziehungen mit Ladenbesitzern. All dies füllt die Taschen angenehm schwer und befreit einen Mann von solch nebensächlichen Sorgen, wie die Bezahlung seiner Leute oder ein Mangel von Segeltuch in den Läden. Erfordert Talent und die selbstbewusste Autorität eines Anführers. (T*0,8+A*0,2) 
}
Accuracy {
 Genauigkeit. Eine der grundlegendsten Eigenschaften für einen Wanderer der endlosen See. Ein echter Kapitän verschwendet kein Pulver und Munition für alle Fische in der Umgebung. Jede Breitseite muss direkt ins Ziel treffen! Und wenn sich der Nebel des Krieges in der salzigen Luft auflöst, dürfen am Horizont keine Trümmer sichtbar sein! Sie sind wertlos ohne gute Einsicht und einen gewissen Grad an Erfolg. (Ei*0,8+Er*0,2) 
}
Cannons {
 Kanonen. Schnell jetzt! Bereiten Sie Ihre Waffen vor und richten Sie sie aus, und bevor Ihre Ohren nicht mehr klingeln, wieder reinigen, wieder laden und zielen. Vergessen Sie nicht, sie mit Wasser zu bespritzen, damit sie nicht überhitzen ... Man darf sich nicht ablenken lassen, in diesem tödlichen Kampf von Donner und Flammen. Ein nur laden, heranrollen und Breitseite abfeuern! Erfordert Talent und Energie. (T*0,6+E*0,4) 
}
Sailing {
 Navigation. Die Fertigkeit, ein Schiff durch Stürme oder Nebel zum Hafen zu bringen. Die Fertigkeit Navigationsgeräte zu lesen und Flotten zu befehligen. In der Hand eines fähigen Kapitäns kann selbst das größte und schreckenerregendste Schiff wie ein Vogel fliegen. Hängt von Einsicht und Talent ab. (E*0,2+T*0,8) 
 Ist die Fähigkeit Navigation nicht ausgebildet, erleidet der Held Abzüge auf alle Fähigkeiten und Charakteristika. 
 Klasse 1 erfordert 95 
 Klasse 2 erfordert 80 
 Klasse 3 erfordert 65 
 Klasse 4 erfordert 40 
 Klasse 5 erfordert 25 
 Klasse 6 erfordert 1 
 Der Unterschied zwischen aktueller und erforderlicher Stufe in der Fähigkeit Navigation wird von der Schiffsklasse abgeleitet. 
}
Repair {
 Reparieren. Wenn der Gegner einen Kugelhagel auf Ihr Schiff niederlässt, das Deck in Sägemehl verwandelt und die Segel in Lumpen, wenn der Regen den Rumpf zerbricht und der Wind die Bindeglieder wie Fasern zerreißt; dies sind die Momente, wenn Reparieren zur wichtigsten Fähigkeit wird. Am wichtigsten ist hier die Ausdauer, aber ohne Einsicht kommt man hier auch nicht weit. (A*0,8+E*0,2) 
}
Grappling {
 Entern. Ein Schiff genau gegenüber dem feindlichen Deck platzieren, eine Planke beim ersten Versuch genau über die Lücke werfen, dann den bibbernden Feind rammen, wie ein Wirbelwind durch die Decks fegen und das Schiff als hochverdiente Prise zu nehmen, all das braucht das gewisse Etwas und Erfahrung. Aber die reichhaltige Belohnung gleicht die ganze Anstrengung aus. Erfordert Reaktion und Talent. (R*0,7+T*0,3) 
}
Defence {
 Verteidigung. Im Laufe der Zeit wird die Bindung zwischen Mannschaft und Schiff stärker bis jeder abgehärtete Seemann mehr wert ist als zehn Frischlinge aus der Schenke. Ein guter Kapitän unternimmt deshalb alles, was in seiner Macht steht, um seinen Männern einen dummen und unnützen Tod im Kampf zu ersparen. Ausdauer und gute Autorität sind beide wichtig, um die Mannschaft für die Verteidigung des Schiffs zu mobilisieren. (E*0,5+A*0,5) 
}
Sneak {
 Heimlichkeit. Durch Gebäude des Gegners schleichen, sich vor Verfolgern im weiten offenen Ozean verstecken oder einem ahnungslosen Gegner wie ein Blitz aus heiterem Himmel zusetzen ist eine Fähigkeit, die kein Abenteurer missen sollte. Unmöglich ohne außergewöhnlichen Erfolg und gute Einsicht. (Er*0,5+Ei*0,5) 
}
Loyality {
 Die Loyalität eines Offiziers wird durch seine Aktionen bestimmt. Offiziere mit einem guten Ruf arbeiten nicht lange für einen Halunken, während Bösewichte kaum Geduld für ein ehrliches Leben haben. 
}
Ship_abilities {
 Fertigkeiten, die die Effektivität bei der Kontrolle eines Schiffs beeinflussen. 
 Die Zahl zeigt, wie viele Fähigkeiten Sie noch verteilt werden müssen. 
}
Personal_abilities {
 Fertigkeiten, die die Fertigkeiten eines Charakters erweitern. 
 Die Zahl zeigt, wie viele Fähigkeiten Sie noch verteilt werden müssen. 
}
Rank {
 Rang - die Entwicklungsstufe eines Charakters im Spiel. Wenn Fähigkeiten steigen, steigt der Rang in Stufen. Wenn der Rang steigt, schreitet auch das Leben des Charakters voran (abhängig von Ausdauer). 
 Steigt ein Rang, lockt der Held stärkere Gegner und höher entwickelte Gegenstände an. 
}
Life {
 Leben - Widerstandsfähigkeit des Charakters in Bezug auf physischen Schaden. Je mehr Lebenspunkte der Charakter besitzt, desto mehr Verletzungen er einstecken kann. 
 Jede Steigerung im Rang verlängert abhängig von seiner Ausdauer sein Leben. 
 }
Health {
 Gesundheit - der allgemeine physische Zustand des Helden. Je schlechter die Gesundheit, desto schwächer sind die Fertigkeiten. Die Gesundheit verschlechtert sich mit weiteren Wunden und regeneriert sich schneller, wenn der Held Nahkämpfen ausweichen kann. 
}
Energy {
 Energie - die Beständigkeit des Charakters im Nahkampf. Alle Angriffsbewegungen kosten eine bestimmte Menge Energie. Wenn ein Charakter im Verlauf des Kampfes ermüdet, braucht er eine Erholungspause, um seine Stärke zurückzuerlangen. 
 Die Energieobergrenze eines Helden hängt von seiner Reaktion ab. 
}
Money {
Geld - Alle lieben Geld, deshalb wird es Geld genannt! Dublonen, Piaster, Taler, Guinee ... Wo liegt der Unterschied? Es ist alles Geld! 
}
Reputation {
 Reputation - Ein Gradmesser dafür, wie andere Sie behandeln. Die Reputation kann hoch (positiv), neutral oder niedrig (negativ) sein. Sie verdienen Reputation durch Ihre Aktionen, was beeinflusst, wie andere Ihre weiteren Aktionen während des gesamten Spiels interpretieren.  
 Der Ruhm eines Helden verblasst mit der Zeit und seine Aktionen verschwinden aus dem Gedächtnis. Die Bösen bleiben immer besser in Erinnerung, während die Guten vergessen werden. Wenn Sie nur untätig herumsitzen, sinkt Ihre Reputation mit der Zeit wieder auf 'normaler Seemann'. 
}
weight {
 Gewicht - das Maximalgewicht, das ein Charakter herumschleppen kann, bevor an Mobilität verliert. 
 Hängt von Kraft und Ausdauer ab. 
 }
Title {
Titel - ein Ehrentitel, der als Kennzeichen dafür dient, wie viel Gutes der Charakter dem Reich gebracht hat, dem er dient.
}
NextExp {
 Die Erfahrungsgrenze zeigt ihren aktuellen Rang, Ansammlung von Fertigkeiten, die erworbenen Fähigkeitspunkte und wie viele Punkte bis zur nächsten Stufe fehlen. Fähigkeitspunkte werden durch Ihre Aktionen verdient.  Zum Beispiel bringt jeder genaue Pistolenschuss, Punkte für ihre Fähigkeit Feuerwaffen. Im Gegenzug werden diese Punkte in ihre globale und persönliche Rangerfahrung oder Schiffsfertigkeiten (abhängig von Fähigkeiten und deren verknüpften Fertigkeiten) geleitet. Wird die Grenze eines Ranges oder einer Fähigkeit überschritten, steigen Sie in die nächste Stufe auf, und erhöhen so ihr Leben und ihre Fähigkeitspunkte, die Sie frei verteilen können.}
TABLE_PERKS {Fertigkeiten des Helden}
TABLE_PERKS_desc {"
 Sie können Ihre Fertigkeiten verteilen, wenn Sie Punkte in Persönliche Fertigkeiten oder Schiffsfertigkeiten haben. 
 Doppelklicken Sie auf die gewählte Eigenschaft, um einen freien Punkt dafür auszugeben. 
 Aufgrund einiger Wechselbeziehungen können verschiedene Eigenschaften zuerst nicht gewählt werden. Zu Anfang müssen Sie die einfachsten Fertigkeiten auswählen.}
 Von da an werden dann fortschrittlichere Fertigkeiten verfügbar. \" {}

PERK_TABLE_NEED_desc {
 Um die gewählte Fähigkeit zu wählen, muss der Held alle folgenden Kenntnisse aufweisen.

Nation_hint {
 Nation des Helden - Die Nation, die Sie zu Beginn des Spiels wählen, oder ansonsten die Nation Ihres aktuellen Patents. Sie können sie im Menü 'Charakter' nachschauen. Im Menü 'Nation' können Sie die aktuelle Flagge Ihres Schiffes sehen, die auf See geändert werden kann, um Ihre wahre Identität zu verschleiern. Aber Feinde des Reichs können auf See entdeckt werden oder durch die Wachen im Hafen. Die Heimlichkeit des Helden beeinflusst den Erfolg der List. Wenn seine gewählte Nation einen Preis auf den Kopf des Helden ausgesetzt hat, dann wird er auch als ihr Feind angesehen. 
 Jede Feindentdeckung berücksichtigt die Beziehungen zwischen den Ländern und die Zahl der Kopfgelder, die auf den Spieler ausgesetzt sind. 
 Schnelle Durchreisen sind in feindlichen Städten nicht möglich. 
}
England_descr {
 England. Wenn irgend jemand Spanien aus der neuen Welt schmeißen kann, dann ist dies England. Während in der alten Welt Frankreich aktiv gegen den spanischen Königshof mit einem weiteren Krieg um den Thron bombardiert, übernimmt England gezielt eine spanische Kolonie nach der anderen, um das größte Seereich der Geschichte zu aufzubauen. 
}
France_descr {
 Frankreich. Seit dem Aufstieg zum Thron hat Ludwig XIV seine militärische Präsenz in der neuen Welt aktiv ausgebaut. Eine große Kolonialmacht, die keine Interessen außer den eigenen akzeptiert. 
}
Holland_descr {
 Die Niederlande. Eine junge Nation, die zur größten Handelsmacht werden möchte. Ein aktiver Verbündeter Frankreichs gegen den gemeinsamen Feind Spanien. In Bezug auf alle anderen ist Holland vollkommen neutral. 
}
Spain_descr {
Spanien. Eine große Kolonialmacht, die den größten Teil der neuen Welt beherrscht. Unter ständigem Druck der beiden anderen Großmächte England und Frankreich, die aktiv ihre eigenen Teile an der neuen Welt vergrößern möchten. 
}
Pirate_descr {
 Piraten - Leute, die keine nationalen Unterschiede anerkennen und nur nach den Gesetzen der Brüder der Küste leben. Das hindert sie aber nicht daran, in Port Royal oder Tortuga herumzuhängen und mit ihrer Beute dort hausieren zu gehen. Sie können sich nach Lust und Laune mit jeder Macht einlassen, doch sind sie vereint in ihrem Hass auf den spanischen Hof. 
 Auf jeden Fall ist eine Piratenflagge für jede Patrouille in den Kolonien so gut wie eine rote. Ein Schurke oder ein Geheimagent? Das ist das Los eines Piraten. 
}
Hunter_hint {
 Ihre Beziehung mit dem Reich durch dessen Soldaten und Bürger. Gute Taten für die Nation erhöhen den internen Ruf, schlechte Taten können einen Silberling auf Ihren Kopf bedeuten. Und das bedeutet eine ewige Jagd über Land und See, denn wenn es in der Karibik eines gibt, dann sind es Kopfgeldjäger. Sie können aber Ihr Kopfgeld durch Ihre Taten oder durch etwas Geld in die richtigen Hände senken. Wenn Sie dem Reich den Rücken kraulen, wird es Ihnen nicht in selbigen schießen. 
}
LevelComplexity {Schwierigkeit}
LevelComplexity_desc {
 Die Schwierigkeit des Spiels reicht in zehn Stufen von 'Seemann' bis 'Unmöglich'. 'Kapitän' ist die mittlere Stufe. Sie beeinflusst den Stufenaufstieg des Charakters, Geld und noch vieles mehr. 
}
RechargePistolOnLine {Pistolen Nachladen}
RechargePistolOnLine_desc {
 In diesem Modus werden Pistolen im Kampf automatisch nachgeladen. Ansonsten bleibt Ihnen nach dem Abfeuern nichts anderes als Ihr Schwert in seiner Scheide. 
 Dieser Modus beeinflusst alle Einwohner der Spielwelt.}
HardcoreGame {Hardcore-Spiel}
HardcoreGame_desc {
 Nicht ausreichend Adrenalin im Spiel? Möchten Sie sich wie ein echter Korsar fühlen, den Rausch der Handlung und Konsequenz zu spüren? In diesem Modus können Sie Ihr Spiel nur an Land in Kirchen speichern. Was für Fehler Sie auch immer machen, Sie müssen damit leben. Sind Sie ein richtiger Kerl? 
}
ArcadeSailMode_desc {
 Die Geschwindigkeit und Dynamik von Seeschlachten. Bevorzugen Sie schnelle Action und viele Aufgaben auf einmal oder langsame taktische Überlegungen? 
}
EncountersRate_desc {
 Die Häufigkeit von Feindbegegnungen im Spiel, insbesondere bezüglich Schiffen auf der Weltkarte. 
}
WorldAlivePause {Im Auftrag des Reiches}
WorldAlivePause_desc {
 Dieser Modus gibt dem Spieler die Möglichkeit, eine der Questszenarien für ein Reich durchzuspielen, friert die 'lebende Welt' aber vor dem Ende des Spiels ein, d. h. Bewegungen von Generalgouverneuren, Invasionen von Städten durch Länder, die Möglichkeit für den Helden, die Stadt zu halten. 
 Die lebende Welt beginnt wieder sich erst wieder zu drehen, wenn das Szenario abgeschlossen ist. 
}
Prisoner {Gefangener}
Prisoner_desc {
 Ein Gefangener kann im Laderaum des Schiffs verhört werden, um seine Identität festzustellen. Dann kann man bei seinem Reich Lösegeld einfordern. 
 Das höchste Lösegeld wird von den Gouverneuren gezahlt, die aus dem gleichen Land wie der Gefangene stammt, auch wenn der Preis in jeder Stadt unterschiedlich sein kann. 
}
shipChoose_hint {
 Klicken Sie auf das Schiff eines Gefährten, um einen freundlichen Handelsmodus für seine Mannschaft zu öffnen. Funktioniert an Land und auf See. Wenn Sie während des Kampfs handeln müssen, müssen Sie direkt neben das Schiff steuern. 
}
GoodsCargo_hint {
 Eine List der Waren im Laderaum des Schiffs. 
 Doppelklicken Sie auf einen Gegenstand, um eine Beschreibung des Gegenstands zu öffnen und eventuell zu löschen. 
}
EXP_SLIDE {Erfahrungszuwachs }
EXP_SLIDE_desc {
 Hier stellen Sie die Geschwindigkeit ein, mit der Ihr Held und seine Offiziere Erfahrung gewinnen. Je höher die Geschwindigkeit, desto schneller werden Fähigkeitenpunkte gesammelt, d. h. neue Stufen und Eigenschaften. 
}
